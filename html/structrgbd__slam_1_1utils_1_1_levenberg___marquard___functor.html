template<typename _Scalar, int NX = Eigen::Dynamic, int NY = Eigen::Dynamic>
struct rgbd_slam::utils::Levenberg_Marquard_Functor< _Scalar, NX, NY >

Structure given to the Levenberg-Marquardt algorithm. It optimizes a rotation (quaternion) and a translation (vector3) using the matched features from a frame to the local map, using their distances to one another as the main metric.
