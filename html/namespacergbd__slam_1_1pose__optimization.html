Implementation of "A General and Adaptive Robust Loss Function" (2019) By Jonathan T. Barron.

Parameters
[in]errorThe error to pass to the loss function
[in]alphaThe steepness of the loss function. For alpha == 2, this is a L2 loss, alpha == 1 is Charbonnier loss, alpha == 0 is Cauchy loss, alpha == 0 is a German MCClure and alpha == - infinity is Welsch loss
[in]scaleStandard deviation of the error, as a scale parameter
Returns
A weight for the current error
