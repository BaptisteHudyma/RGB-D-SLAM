Searches for a match in the detectedFeatures object.

Parameters
[in]detectedFeaturesThe object that contains the detected features for this frame
[in]worldToCameraThe matrix to convert from map space to camera space. This should be an estimate of the position from which detectedFeatures were observed
[in,out]isDetectedFeatureMatchedA vector of booleans indicating which detected features are already matched. should be updated if a match is found
[in,out]matchesThe object that contains the feature matches
[in]shouldAddToMatchesIf false, will mark feature as matched but wont add them to the match vector
[in]useAdvancedSearchIf true, will search matches with a lesser accuracy, but further
Returns
the index of the match if found, or UNMATCHED_FEATURE_INDEX

Implemented in rgbd_slam::map_management::MapPoint2D, rgbd_slam::map_management::MapPoint, and rgbd_slam::map_management::MapPlane.
