Apply each plane and cylinder mask on the maskImage, with colors corresponding to the ids of colors vector.

Parameters
[in]inputImageinput RGB image on which to put the planes and cylinder masks
[in]colorsA vector of colors, that must remain consistant at each call
[in]maskImageAn image where each pixel is the index of a plane or cylinder
[in]primitiveSegmentsA container of the planes and cylinders detected in inputImage
[in,out]labeledImageThe final result: an image with colored masks applied for each plane and cylinder, as well as a bar displaying informations on the top. Must be passed as an empty image of the same dimensions as labeledImage.
[in]elapsedTimeThe time elapsed since last frame. Used to display fps
[in]associatedIdsA map associating each plane/cylinder index to the ids of last frame version of those planes/cylinders
